# Life Design — веб-прототип

Интерактивный прототип главного экрана платформы «Дорожная карта будущего личности человека».

## Как запустить

Просто откройте `index.html` двойным кликом в браузере. Никакой установки и сборки не требуется.

> Для лучшего рендеринга используйте Chrome, Edge или Firefox.

Карта полностью адаптирована под основу из `mainDocs` (источник истины). Канон дизайна — философия «текучесть времени и пространства» (`mainDocs/_extracted/site_philosophy_tekuchest_vremeni.txt`) и состав финальной дорожной карты (`mainDocs/_extracted/итоговый продукт.txt`). Тон — премиум 35+ (без геймификации).

## Что внутри

**Карта пространства-времени** (метафора чёрной дыры):
- Центральный хаб «Вы здесь · точка А» — настоящее, фигура света (вне возраста/пола) на платформе с пульсирующими кольцами
- **3 орбиты-сценария основы по интенсивности перехода** (радиус = смелость): Бережный (бирюзовый) · Рост (фиолетовый) · Пересборка/Прорыв (янтарный)
- Вдоль каждой орбиты — 6 узлов на **контрольных точках** 7д / 30д / 3мес / 6мес / 1год / 2–3года, с ролью-ветвью на дальнем горизонте
- **Гравитационные поля прошлого** под хабом (Привычная роль · Страх перемен · Обязательства — тянут назад), деликатно
- **Точки невозврата** (горизонт событий) — особые узлы-ромбы с оранжевым свечением и пояснением «момент пересборки, а не катастрофа»
- **Прототипы** (силы ускорения) — узлы-проверки спроса (экспертный пост, пробная консультация, 7-дн эксперимент)
- Зоны Прошлое / Настоящее / Будущее; **программная искривлённая сетка пространства-времени** вместо статичного PNG
- Hover-тултипы (горизонт + шаг), модалка с шагами и этическим дисклеймером

**Боковые панели (состав дорожной карты основы):**
- Слева: точка А + «Готовность к переходу», пространства жизни, **карта идентичности** (был→есть→становлюсь), ценности, **социокультурный код** (ресурс/ограничение), профессиональный капитал
- Справа: текущий фокус, **новая роль/позиционирование** (по выбранной орбите), **модель дохода** (качественно, без % обещаний), прототипы будущего, контрольные точки, ритм действий

**Шапка:** бренд «Дорожная карта будущего» (дескриптор Life Design), прогресс по 6 этапам методологии, профиль без XP/уровней (Этап N/6).

**Низ:** горизонт движения 2–3 года (таймлайн контрольных точек) + ключевые индикаторы (Ясность, Шаги, Прототипы, Окружение, Доходные гипотезы — без числового прогноза дохода).

**Этика:** постоянный дисклеймер в подвале и в модалках («карта помогает выбирать, но не выбирает за вас; не прогноз и не диагноз»).

> Прежняя версия (4 трека-направления, геймификация XP/streak/кристаллы, фон-PNG) заменена. Фоновые `map-*.png` картой больше не используются.

## Регенерация AI-ассетов

Фон карты и фигуру генерирует Grok (`grok-imagine-image`). Ключ берётся из `.env` (`XAI_API_KEY`):

```powershell
node tools/generate-assets.mjs          # фон + фигура
node tools/generate-assets.mjs figure   # только фигура
node tools/generate-assets.mjs bg       # только фон
```

Промпты правятся в скрипте. Фигура автоматически получает прозрачный фон (alpha по яркости через `jimp`; при отсутствии — `npm i jimp@0.22`). Прежний `generate-map-images.mjs` оставлен для истории, но карта его PNG больше не использует.

## Управление (index.html)

- Клик на ноду → модальное окно с шагами и дисклеймером (для точки невозврата — пояснение «момент пересборки»)
- Клик на орбиту-сценарий (Бережный / Рост / Пересборка) → выделяет орбиту и запускает анимированный поток времени вдоль неё
- Движение мыши над картой → лёгкий параллакс слоёв (фон и фигура смещаются на разной глубине)
- Esc → закрывает модальное окно

> Карта — плоская (2D). Пробовали 3D-перспективу (CSS-наклон) и полноценный WebGL/Three.js — оба варианта отклонены, оставлен плоский вид с параллаксом.

## Адаптив (один экран на ПК)

- **ПК (≥1024px):** вся раскладка помещается в один экран без прокрутки страницы — `h-screen` + flex-строка: компактные сайдбары со внутренним скроллом (`thin-scroll`) ↔ карта-центр на всю высоту (`flex-1`) ↔ нижняя полоса (горизонт + индикаторы + дисклеймер).
- **Планшет (640–1023px):** вертикальный поток — карта сверху, карточки сайдбаров в сетке 2 колонки.
- **Телефон (<640px):** вертикальный поток в 1 колонку; фигура-хаб масштабируется под размер карты через `.hub-scale` (медиазапросы). Порядок на мобиле: карта → полоса → левый блок → правый блок (через `order-*`).

## Стек

- React 18 (через CDN)
- Tailwind CSS (через CDN)
- Babel Standalone для in-browser JSX
- SVG для путей и иконок
- Inter / Manrope шрифты

Это сознательный выбор для MVP — нулевая настройка. На следующей итерации мигрируем на Vite + TypeScript + React Router когда понадобится бэкенд, авторизация и роутинг.
