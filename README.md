# Дорожная карта будущего · Life Design

Интерактивный прототип «карты пространства-времени» — личная траектория развития на 2–3 года в метафоре чёрной дыры: **точка А** в ядре, три **орбиты-сценария** (Бережный · Рост · Пересборка), контрольные точки на горизонте 7 дней → 2–3 года, гравитация прошлого и точки невозврата.

🔭 **Живое демо:** https://legodev94.github.io/dorozhnaya-karta-budushchego/

## Страницы

- **`index.html`** — интерактивная карта (личный кабинет): орбиты-сценарии, кликабельные узлы, искривление гравитацией прошлого, поток времени, адаптив (один экран на ПК, мобильная вёрстка).
- **`landing.html`** — продающий лендинг: три формата (Диагностика → Alive Roadmap → Premium), «что вы получите», отличие от коучинга, FAQ.

## Стек

React 18 + Tailwind CSS (через CDN), Babel Standalone, SVG. Один файл на страницу, без сборки — открывается двойным кликом. Деплой на GitHub Pages из ветки `main`, папка `/docs`.

Подробности по дизайну и управлению — в [`docs/README.md`](docs/README.md).

---

*Прототип продукта. Карта помогает выбирать, но не выбирает за вас — это не прогноз, не диагноз и не финансовая/медицинская/психологическая услуга.*
